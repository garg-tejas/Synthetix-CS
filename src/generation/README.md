# Answer Generation Module

This module will implement Phase 2: Answer Generation with Citations.

## Planned Components

- `context_builder.py` - Format retrieved chunks into LLM-ready context with citation markers
- `generator.py` - Generate answers from context using LLM
- `citations.py` - Extract and validate citations from generated answers
- `prompts.py` - Prompt templates for answer generation
